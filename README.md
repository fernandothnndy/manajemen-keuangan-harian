LAPORAN PROGRAM PERCABANGAN DAN PERULANGAN
STUDI KASUS “MANAJEMEN KEUANGAN HARIAN”

Program Manajemen Keuangan Harian adalah sebuah sistem yang dirancang untuk membantu pengguna dalam mengelola, memantau, dan mengontrol penggunaan dana sehari-hari secara terstruktur dan efisien. Program ini memungkinkan pengguna mencatat aktivitas penggunaan pemasukan dan pengeluaran, melihat riwayat transaksi, serta dapat memberikan saran jika sudah tidak terkontrol.
Melalui program ini, pengguna dapat menambahkan, mengecek, dan mengelola dana yang diperoleh dan digunakan. Program ini juga menyediakan saran yang membantu pengguna mengevaluasi pola penggunaan pendapatan harian.
Berikut adalah algoritma beserta flowchart dari program manajemen keuangan harian.
1.	Algoritma dan flowchart
-	Program dijalankan
-	Program menampilkan menu utama kepada user:
a.	Tambah pemasukan
b.	Tambah pengeluaran
c.	Lihat saldo
d.	Riwayat transaksi
e.	Selesai/done
-	Pengguna diminta memasukkan angka (1-5)
Proses berdasarkan pilihan menu
-	Jika pengguna memilih angka 1 (tambah pemasukan)
1.	Program meminta user memasukkan kategori pemasukan (contoh: gaji, uang jajan, bonus, hadiah)
2.	User memasukkan kategori pemasukan
3.	Program meminta user memasukkan jumlah pemasukan
4.	User memasukkan nominal pemasukan
5.	Program:
Menambahkan jumlah pemasukkan ke saldo
Menyimpan data pemasukan ke Riwayat transaksi
6.	Program menampilkan pesan “Pemasukan berhasil ditambahkan”
7.	Program Kembali ke menu utama

-	Jika pengguna memilih angka 2 (tambah pengeluaran)
1.	Program meminta user memasukkan kategori pengeluaran (contoh: makan, ongkos)
2.	User memasukkan kategori pengeluaran
3.	Program meminta user memasukkan jumlah pengeluaran
4.	User memasukkan nominal pengeluaran
5.	Program:
Mengurangi saldo dengan jumlah pengeluaran
Menyimpan data pengeluaran ke Riwayat transaksi
6.	Program menampilkan pesan “Pengeluaran berhasil ditambahkan”
7.	Program Kembali ke menu utama

-	Jika user memilih angka 3 (Lihat saldo)
1.	Program menampilkan
Saldo saat ini
Saran sederhana
2.	Program kembali ke menu utama

-	Jika user memilih angka 4 (Riwayat transaksi)
1.	Program menampilkan seluruh Riwayat pemasukan dan pengeluaran
2.	Program Kembali ke menu utama

-	Jika user memilih angka 5 (done)
1.	Program akan menampilkan pesan “terima kasih telah mengguakan program”
2.	Program dihentikan

-	Selesai
